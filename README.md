# 💳 Bradesco Recomenda

Chatbot inteligente de recomendação de produtos financeiros desenvolvido como solução para o desafio de laboratório da [DIO](https://www.dio.me/).

## 📌 Sobre o projeto

O **Bradesco Recomenda** é um assistente virtual que utiliza IA generativa para recomendar produtos financeiros personalizados ao usuário — como cartões de crédito, seguros, investimentos e outros serviços do Bradesco — com base no perfil e nas necessidades informadas durante a conversa.

## 🚀 Tecnologias utilizadas

- **Python** — linguagem principal da aplicação
- **IA Generativa** — modelo de linguagem para geração de respostas e recomendações
- **Prompt Engineering** — estruturação de prompts para guiar o comportamento do chatbot

## 🧠 Como funciona

1. O usuário inicia uma conversa com o chatbot
2. O assistente coleta informações sobre o perfil financeiro do usuário
3. Com base nas respostas, a IA gera recomendações personalizadas de produtos Bradesco
4. O usuário recebe sugestões explicadas de forma clara e acessível
